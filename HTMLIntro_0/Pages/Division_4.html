div, Division kelimesinin kısaltılmısı olarak HTML dünyasına giriş yapmıstır...Anlamı, tasarımsal alan demektir. Birbirinden bagımsız görsel acıdan cok daha iyi ve kontrol edilmesi kolay bir tasarım alanı sunar...Özellikle div tag'i hyatımıza girmeden önce tablolar ile yapılan tasarımlar hem performans acısından sıkıntılıydı hem de zor bir görsel tasarım işlemi yapmamıza sebep oluyordu

Bazı temel kavramlar

  • id => Site üzerinde yer alan bir tag'i ifade etmek icin kullanılan attribute'tyr...Bir ID benzersiz olmalıdır...Ancak zorlarsanız aynı Id'yi baska elementlere de verebilirsiniz...Tabii ki bu durum W3C kurallarına aykırıdır.Kullanılan bir Id aynı sayfadaki baska bir Html elementi tarafından kullanılmamalıdır...
  • width => Bir elementin genişligini ifade eder... Bazı elementlerde tek basına bir attribute olarak yazılabilse de bazı elementlerde bu tanım CSS ister...
  • heigth => Bir elementin yüksekligini ifade eder... Bazı elementlerde tek basına bir attribute olarak yazılabilse de bazı elementlerde bu tanım CSS ister...
  • margin - left/top/bottom/right => Bir elementin ilgili taraftan dıs cercevesine olan uzaklıgını temsil eder
  • padding - left/top/bottom/right => Bir elementin ic cercevesinde olanların elementin kendi kenarına olan uzaklıgını belirler
  • style - bir css yazabileceginizi belirten attribute'tur....Biz bu tarzda yazımlara inline css deriz...
